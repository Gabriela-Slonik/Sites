Hash Encadeada

A técnica de Hash Encadeada (ou hashing com encadeamento separado) utiliza listas ligadas para resolver colisões em uma tabela hash. Cada posição da tabela contém uma lista onde são inseridos os elementos que geram o mesmo valor de hash. Essa abordagem é simples e eficiente quando o número de colisões é controlado, mantendo a busca próxima de O(1).
